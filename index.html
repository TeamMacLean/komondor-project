Research, Collaborate, Win!

Komondor is a place to store, share and collaborate your scientific data

Why

  • Datahog needs to be replaced.
  • Irida needs work.
  • First class Galaxy integration.
  • Support different backends (Object Store, S3).
  • Auto visualization.
  • Better privacy controls.
  • Collaboration.
  • Wiki.
  • Resumable uploads.
  • Support for new ENA API.
  • Tagging.
  • Search.
  • Modular everything.
  • Easy file conversion ('Download as').
  • Image support (maybe).
  • LDAP, Social, Github and Local user support.

How

  • Micro-Services.
  • GraphQL API.
  • Mongo DB.
  • Official library.
  • NextJS Progressive Web App.
  • TUS upload protocol.
  • Datahog structure.
  • Direct upgrade from Datahog.
